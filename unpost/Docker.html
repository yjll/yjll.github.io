离线安装镜像

先从一个有网络的电脑下载docker镜像
docker pull centos
保存镜像到本地文件
docker save -o centos_image.docker centos
把镜像拷贝到无网络的电脑，然后通过docker加载镜像即可。
docker load -i centos_image.docker
